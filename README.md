Цель работы
Цель практической работы — освоить процесс отладки программного обеспечения с использованием встроенных инструментов среды разработки Microsoft Visual Studio, закрепить навыки выявления и устранения типичных ошибок в коде.

Как найти задания
Каждое задание размещено в отдельной ветке репозитория. Для доступа к нужному заданию необходимо переключиться на соответствующую ветку.

Вывод по работе
В ходе выполнения практической работы были изучены и применены ключевые инструменты отладчика Microsoft Visual Studio:

Точки останова (обычные, условные и временные) — позволяют приостановить выполнение программы в заданной точке кода, чтобы проанализировать состояние приложения.

Окна наблюдения («Локальные», «Авто», «Контрольные значения») — предоставляют детальную информацию о значениях переменных на текущем этапе выполнения программы.

Пошаговое выполнение (Step Into, Step Over, Step Out) — даёт возможность детально отследить последовательность выполнения инструкций и выявить участки с некорректной логикой.

Стек вызовов (Call Stack) — помогает понять цепочку вызовов методов, приведшую к текущему состоянию программы, и определить источник ошибки.

DataTips — позволяет быстро просмотреть значения переменных прямо в редакторе кода без переключения на окна наблюдения.

С помощью этих инструментов удалось обнаружить и исправить следующие ошибки:

Исключение IndexOutOfRangeException в приложении «Фибоначчи» — вызвано некорректной инициализацией списка: размер коллекции не соответствовал ожидаемому, что приводило к выходу за границы массива при обращении к элементам. Ошибка устранена путём проверки границ и корректной настройки размера списка.

Опечатка в switch-конструкции приложения «Галактики» — из‑за визуального сходства символов 'l' (строчная L) и 'I' (заглавная i) выполнялась неверная ветвь кода. Проблема решена путём тщательной проверки и замены ошибочного символа.

Полученные навыки отладки позволят в дальнейшем эффективнее разрабатывать и тестировать программные продукты, сокращая время на поиск и устранение дефектов.
