Задание 1
Консольное приложение вычисляет n-е число Фибоначчи итеративным способом.

При первом запуске программа выполняется без ошибок, но возвращает неверный результат.

Ошибки, обнаруженные при отладке
Ошибка 1 — неверный результат вычисления

Где: метод Fibonacci, строка for (int i = 2; i < n; i++)
Причина: цикл выполняется до i < n, из-за чего последняя итерация пропускается
→ фактически вычисляется Fibonacci(n - 1)
Исправление: заменить условие на i <= n
Как был найден баг
Запустил приложение с отладчиком (F5). Программа завершилась без исключений, но вывела значение 3 вместо ожидаемого 5.

Далее:

поставил точку останова внутри цикла
отслеживал значения переменных в окне «Локальные»
заметил, что цикл заканчивается при i = 4, хотя должен выполниться ещё один шаг
Перед выходом из метода значения были:

n1 = 2
n2 = 3
Это соответствует Fibonacci(4), а не Fibonacci(5).

Использованные способы отладки
Инструмент	Применение
Точка останова (F9)	Поставлена внутри цикла и перед return
Окно «Локальные»	Отслеживал n1, n2, sum, i, n
Подсказки по данным (DataTips)	Просмотр значений переменных при наведении
Шаг с заходом — Step Into (F11)	Вход в метод Fibonacci
Шаг — Step Over (F10)	Переход по итерациям цикла
Continue (F5)	Быстрый переход между итерациями
Результат после исправления
for (int i = 2; i <= n; i++)
